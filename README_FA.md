# 🚀 MasterDNS Toolkit

ابزاری برای استخراج، مرتب‌سازی و تست سرعت IPها در بستر **MasterDNS VPN** با قابلیت اتوماسیون کامل.

---

## 📌 معرفی

این اسکریپت برای انجام عملیات‌های زیر طراحی شده است:

* استخراج IP از فایل‌های مختلف
* مرتب‌سازی و حذف IPهای تکراری
* تست اتصال و سرعت هر IP از طریق SOCKS5
* مدیریت و آماده‌سازی فایل‌های تنظیمات
* تبدیل فایل‌های خروجی برای تحلیل بهتر

---

## ⚙️ پیش‌نیازها

قبل از اجرای اسکریپت، موارد زیر باید روی سیستم شما نصب باشد:

```bash
sudo apt install curl netcat bc
```

---

## 🛠 نصب و راه‌اندازی

### 1. دانلود اسکریپت

```bash
git clone https://github.com/your-repo/MasterDNS_Toolkit.git
cd MasterDNS_Toolkit
chmod +x MDNS_Toolkit.sh
```

---

### 2. تنظیم فایل config

در اولین اجرا، فایل زیر به‌صورت خودکار ساخته می‌شود:

```
MasterDNS_tool.cfg
```

در صورت نیاز آن را ویرایش کنید:

```bash
nano MasterDNS_tool.cfg
```

---

### ❗ نکته مهم

حتماً نام فایل اجرایی را مطابق سیستم خود تغییر دهید:

```bash
EXECUTABLE="./MasterDnsVPN_Client_Linux_AMD64_v2026.03.12.145701-00150d5"
```

🔺 این مقدار باید دقیقاً برابر با نام فایل واقعی کلاینت شما باشد.

---

## 🚦 قبل از اجرا

سرویس MasterDNS باید متوقف شود:

```bash
sudo systemctl stop masterdnsvpn
```

---

## 📂 ساختار فایل‌ها

| فایل                     | توضیح               |
| ------------------------ | ------------------- |
| `sorted_ip.txt`          | لیست IPهای مرتب شده |
| `client_resolvers.txt`   | IP در حال تست       |
| `output/`                | خروجی‌ها و لاگ‌ها   |
| `client_config.toml`     | تنظیمات کلاینت      |
| `client_config.original` | بکاپ تنظیمات        |

---

## 🧠 نحوه عملکرد اسکریپت

### مرحله 1: استخراج IP

IPها از فایل‌های ورودی خوانده می‌شوند و فقط IPهای معتبر نگه داشته می‌شوند.

### مرحله 2: مرتب‌سازی

IPها به‌صورت عددی مرتب شده و تکراری‌ها حذف می‌شوند.

### مرحله 3: تست اتصال

برای هر IP:

1. اجرای کلاینت MasterDNS
2. بررسی فعال شدن SOCKS5
3. ارسال درخواست تست (3 بار)

### مرحله 4: تست سرعت

در صورت موفق بودن اتصال:

* دانلود تست از Cloudflare انجام می‌شود
* سرعت (KB/s) و زمان ثبت می‌شود

### مرحله 5: ثبت لاگ

تمام نتایج در فایل زیر ذخیره می‌شوند:

```
output/speed_DATE.log
```

---

## 🧪 دستورات (Commands)

### 🔹 استخراج IP از لاگ‌ها

```bash
./MDNS_Toolkit.sh sa
```

---

### 🔹 مرتب‌سازی IP از فایل

```bash
./MDNS_Toolkit.sh s file.txt
```

---

### 🔹 تست سرعت

```bash
./MDNS_Toolkit.sh st
```

یا برای یک IP خاص:

```bash
./MDNS_Toolkit.sh st 1.2.3.4
```

---

### 🔹 تبدیل فایل‌های txt به log

```bash
./MDNS_Toolkit.sh conv
```

📌 همه فایل‌های `.txt` به `.log` تبدیل می‌شوند به‌جز:

* `client_resolvers.txt`
* `sorted_ip.txt`

---

### 🔹 آماده‌سازی client_config

```bash
./MDNS_Toolkit.sh prep
```

✔ این دستور:

* از فایل بکاپ می‌گیرد
* مقادیر بهینه را اعمال می‌کند

---

### 🔹 همگام‌سازی تنظیمات

```bash
./MDNS_Toolkit.sh sync
```

✔ این دستور:

* اطلاعات SOCKS را از `client_config.toml` می‌خواند
* در `MasterDNS_tool.cfg` اعمال می‌کند

---

## 📊 خروجی‌ها

* فایل مرتب‌شده:

```
sorted_ip.txt
```

* نسخه بکاپ با تاریخ:

```
output/sorted_ip_YYYY-MM-DD_HH-MM-SS.txt
```

* لاگ تست سرعت:

```
output/speed_YYYY-MM-DD_HH-MM-SS.log
```
